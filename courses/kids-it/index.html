Certificate in Fundamentals of Computing - Level 1

Overview
Certificate in Fundamentals of Computing - Level 1 course provides the foundation for the use of computers and computer softwares in day to day life.


16 Weeks

Duration

LKR 13,000

Course Fee

20 Students

Group Size



Contents

Importance of computers

  • Functions of the computer and its characteristics
  • Identifies computer components
  • Describes the need for software
  • Appreciates the use of a computer to carry out routine work

Use computers safely

  • Uses the computer laboratory properly
  • Use computers and peripherals safely

Organization of the Computer

  • Identifies the components of a CPU
  • Describes the evolution of computers

Operating Systems

  • Describes the different types of operating systems
  • Identifies computer storage as a collection of digital data on different media
  • Creates folders Save/open/edit/ delete/ re-name/ copy/move/ files
  • Handling files
  • Explores file properties

Use computers efficiently and effectively with operating system

  • Uses Operating System for configuring and applying settings
  • Explores basic troubleshooting of computers and maintenance procedures (hardware and software )

Use application softwares appropriately

  • Uses text editing and drawing software effectively
  • Uses audio software and video software effectively

Uses text editing software to type effectively

  • Uses computers efficiently by developing typing skills

MS Office Word

  • Uses basic functions of MS Word in creating a document

MS Office PowerPoint software to create presentation

  • Uses basic functions of Presentation software in creating a presentation

MS Office Excel software for calculations and for simple analysis of data

  • Describes basics of MS Excel software
  • Enters data in worksheet
  • Carries out Simple mathematical calculations

Internet for information gathering and communication

  • Identifies the Internet as a collection of information resources in a shareable pool
  • Uses search engines to obtain information
  • Uses computers and data safely and securely

Uses the services of the Internet and develop web pages

  • Uses resources available in the Internet (text, images, audios, videos etc.)
  • Uses offline (E-mail) and online (Chat) Communication
  • Develop web pages using HTML
  • Uses the Internet safely, securely and ethically
