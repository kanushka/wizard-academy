Certificate in Fundamentals of Computing - Level 2

Overview
Certificate in Fundamentals of Computing - Level 2 course is beyond the basics and provides more knowledge on computing.


16 Weeks

Duration

LKR 14,000

Course Fee

20 Students

Group Size



Contents

MS Office Word

  • Use basic functions of Word software in creating a document
  • Compiling a professional document(ex: Thesis)

MS Office PowerPoint software to create presentation

  • Uses basic functions of Presentation software in creating a presentation

MS Office Excel software for calculations and for simple analysis of data

  • Describes basics of MS Excel software
  • Enters data in worksheet
  • Carries out Simple mathematical calculations
  • Uses Functions to carry out Simple mathematical calculations and data sorting
  • Uses various charts to display data

Flow charts for problem solving

  • Analyze simple problems by decomposing and connecting them logically
  • Use Sequence, Selection and Iteration control structure for drawing flow charts
  • Use Selection and Iteration (Repetition) control structures for solving simple
  • Evaluate the solution to ensure that it properly satisfies the problem

Develop programs using Scratch(https://scratch.mit.edu/about)

  • Develop simple programs using visual development environment
  • Describe the concept of variables in programs
  • Develop programs with visual support with nested iterations
  • Develop programs with array variables
  • Identifies errors in a program

Physical Computing

  • Programs simple digital systems using microbit(https://makecode.microbit.org/)
  • Use a simple hardware device to implement physical computing

Network and Internet

  • How internet works
  • Use resources available in the Internet (text, images, audios, videos etc.)
  • Use the Internet safely, securely and ethically
  • Search information on the internet
  • Resource sharing and communicating network

Web page development

  • Develop web pages using HTML, CSS, JavaScript
