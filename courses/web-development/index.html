Certificate in Web Development

Overview
Introduction to Web Development is a one-stop course that covers all of the tools you’ll need to create websites. The course contains in-depth discussions of each of “The Big Three” tools used for Website development: Hypertext Markup Language (HTML), Cascading Style Sheets (CSS), and JavaScript as well as getting started with GitHub and PHP. This course gives you the skills you need to be a professional web developer.
